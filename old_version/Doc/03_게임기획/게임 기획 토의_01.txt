게임 설명

장르 : 턴제 RPG

세계관 : 중세 판타지

캐릭터 : 5~6 등신


요소
m(미터) 단위 사용

캐릭터 마다 부여받는 턴
한번에 모든 캐릭터의 행동을 지시해 놓고 턴에 따라 움직이는 것이 아님
	- 엑스컴 키메라 스쿼드
	- 포켓몬

한 라운드에 모든 캐릭터가 최소 한번은 행동한다.

턴에 할 수 있는 행동
	- 대기
	- 이동
	- 카드 사용
		- 스킬  - 공격 / 방어 / 버프 / 강화
			메인 행동
				공격 / 방어 / 버프
			서브 행동
				강화
			
		- 카운터

대기 -> 이동 -> 카드사용
순서로 행동할 수 있다.
만일 다음의 행동을 했다면 이전단계의 행동을 할 수 없다.

대기
	내 턴을 가장 나중으로 미루는 행위
	만일 여러 캐릭터 대기를 했다면 먼저 대기한 캐릭터가 먼저 행동한다

이동
	캐릭터가 다른 위치로 이동한다.

카드 사용
	손패에 있는 카드들을 사용한다.
	카드들의 사용 횟수 제한은 없다.

턴 종료 조건
	- 대기를 한다.
	- 공격 카드를 사용한다.
	- 직접 턴 종료를 한다.


콤보 시스템
메인이 되는 행동을 하게 되면 (공격 / 방어 / 버프) 콤보 UI 가 열린다.
해당 UI에 강화 카드를 추가하여 메인 행동의 효과를 상승 시킬 수 있다.
강화에 사용된 카드의 갯수에 따라 이펙트가 변할 수 있다.


카르마(가제) 시스템
카드를 사용하면 카르마가 올라간다.
적 Ai는 가장 높은 카르마를 가진 캐릭터를 우선 공격한다.
	제안 1
	모든 캐릭터는 가지고 있는 카르마만큼 추가 데미지를 받는다.
	제안 2
	각 플레이어 마다 가장 높은 카르마를 가진 캐릭터는 추가 카르마 데미지를 받는다.
카르마는 정확한 수치로 표현하지 않지만 비교는 가능하도록 UI 표기를 한다.
카르마 상승 방법
	- 스킬을 사용하면 해당 범위 내의 모든 오브젝트에 데미지를 주어
	  해당 수치에 비례하여 카르마를 상승시킨다.


마을 -> 던전(인스턴트 던전)

던전은 몇개의 스테이지로 구성

죽은 캐릭터는 회복 계열 캐릭터의 부활 스킬로 부활 가능
- 부활 스킬은 높은 카르마 코스트를 가짐 ( 또는 죽은 캐릭터의 카르마를 계승함 )

플레이어가 조종하는 캐릭터
1 ~ 4

직업 시스템
캐릭터 마다 직업이 있다.
직업마다 사용하는 카드가 다르다
공용이 있을 수는 있다.
직업 별 사용할 수 있는 카운터 카드가 다르다

카운터
카운터 카드는 일종의 고유기 이다.
카운터 카드는 마을에서 만들 수 있다.
카드 형식은 카운터 로 고정
범위 / 데미지 / 버프 / 디버프 등등의 효과를 직접 지정해서 만들 수 있는 카드
각 캐릭터 마다 1개씩 만 가질 수 있다.
카운터 카드를 만드는 방법은
플레이 도중에 기본이 되는 카운터 카드를 얻을 수 있고
마을에 돌아가서 해당 카드의 세부수치를 조정하여 고유기를 만들 수 있다.
특정 카드의 경우 특정한 세부수치가 수정할 수 없을 수도 있다.

범위 - [원형] 10m
사거리 - 0m
데미지 - 100
버프 - 이번턴 공격력 1000%
디버프 - 이동 불가
리스크 - 전투 불능
지속시간 - 0
사용 횟수 - 1


메인이 되는 카운터 카드 자체가 일정 포인트를 여분으로 가진다.
각 수치 별로 소모하는 포인트 가 있어서
해당 수치를 소모하여 스킬의 효과를 변경한다,

범위
	원형
		반지름
	부채꼴
		각
		반지름
	선형
		두께
		길이
	대상
		몇명

사거리
	~m

데미지
	(n, m) ~ (n+a , m+a) - 중 n,m,a 값을 조정할 수 있다.

버프
	부활
	스탯 상승
		이동 가능 거리
		속도
		카드 효과
		체력
	카르마 감소

디버프
	체력
	스탯 감소
		버프와 동일
	CC (군중 제어)

리스크
	카드 사용 불가
	이동력 감소
	스탯 감소
	전투불능
	카르마 상승량 증가
	...

지속시간
	라운드 단위

사용 횟수
	던전 단위로 사용할 수 있는 횟수
	스테이지 단위로 사용할 수 있는 횟수
	라운드 단위로 사용할 수 있는 횟수
	턴 단위로 사용할 수 있는 횟수


(미정 - 안할수도 있음) 에니메이션을 플레이어가 만들 수 있게


카드 종류에 대한 설명

공격 / 방어  / 버프 / 강화 / 카운터

공격
	한 턴에 한번만 사용 할 수 있는 카드
	사용을 하면 턴이 종료 된다.
	입힌 피해만큼 카르마가 상승한다.

방어
	방어도(보호막)을 쌓는다.
	가장 높은 수치만 적용된다.
	지속시간은 무제한

버프
	대상에게 주는 버프

강화
	카드의 효과를 강화하는 카드

카운터
	상대의 행동에 대한 리액션으로 사용할 수 있는 카드
	특정 조건을 만족했을 때만 사용할 수 있는 카드
	대신 강력한 효과를 가진 카드
	뽑아서 사용하는 것이 아닌 모든 전투 시작 시 손패에 들고 시작하는 카드
	던전당 사용 횟수가 정해져 있는 카드
	사용에 턴 제약이 없다.


던전(전투가 일어나는) 내 오브젝트 종류

플레이어 캐릭터
	내 캐릭터들 (4)
중립 오브젝트
	지형 위에 올라가 있는 모든 물체
		건물
		바위
		나무
		...
적 오브젝트
	에너미 캐릭터


게임의 진행 순서
메인 화면 -> 마을 -> 던전 -> 스테이지 -> 전투 -> 마을

전투 -> 라운드(전투가 끝날 때 까지 반복)



라운드 분해

셋업

	속도에 따라 캐릭터의 순서 결정

메인
	각 캐릭터 턴 진행

클린업
	각 캐릭터 지형 효과 적용
	라운드 지속 효과에 대한 효과 처리
	라운드 지속 효과에 대한 지속시간 감소 처리
